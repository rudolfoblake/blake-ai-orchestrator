# Código de Conduta

Este projeto adota o **Contributor Covenant v2.1** como base para seu código de conduta.

Links oficiais:

- Versão em inglês: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- FAQ: https://www.contributor-covenant.org/faq

## Compromisso

Nos comprometemos a criar um ambiente aberto e acolhedor. Não toleramos assédio ou discriminação de qualquer natureza.

## Padrões

Exemplos de comportamento que contribuem para um ambiente positivo:

- Usar linguagem acolhedora e inclusiva.
- Respeitar diferentes pontos de vista e experiências.
- Aceitar críticas construtivas de forma educada.

Exemplos de comportamento inaceitável:

- Uso de linguagem ou imagens sexualizadas; assédio.
- Comentários ofensivos; ataques pessoais ou políticos.
- Perturbação contínua de discussões.

## Escopo

Este Código se aplica em todos os espaços do projeto, incluindo issues, PRs, discussões e eventos relacionados.

## Execução

Se você observar comportamento abusivo, por favor **abra uma issue** com o rótulo `conduct` descrevendo o ocorrido (inclua contexto e evidências quando possível). Caso prefira, você pode sinalizar diretamente ao mantenedor no GitHub (`@rudolfoblake`).

Todas as denúncias serão revisadas de forma rápida e justa. A equipe do projeto pode tomar medidas, inclusive aviso formal, restrição de participação ou banimento.
